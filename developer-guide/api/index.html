API - Backend

Requirements

{
  "node": ">=0.12",
  "mongodb": ">=2.6"
}

Set up your environment

Install prerequisites

Install MongoDB: Offical Guide

# Update npm if you haven't already
npm install -g npm

# Install loopback
npm install -g strongloop

Install npm deps

# cd into api
cd api

# npm deps
npm install

Useage

# start mongodb daemon (in separate window/tab)
mongod

# run
node .
